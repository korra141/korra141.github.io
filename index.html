Thesis

Differentiable Bayesian Filtering with Harmonic Exponential Distributions

Robots navigating without GPS rely on noisy, context-dependent sensors whose measurement distributions classical filters can't accurately model. This work introduces Diff-HEF, an end-to-end differentiable filter that learns calibrated observation likelihoods as non-parametric Harmonic Exponential Distributions on Lie groups. Trained via backpropagation through time, it produces better-calibrated pose estimates than EKF, Histogram, and particle filter baselines.

LLM · 2026

LLM's understanding of Numerical Reasoning

How well do LLMs actually understand numbers? This project probes that question through financial forecasting on major stocks, benchmarking FinGPT against ChatGPT, LLaMA 2 & 3, and FinBERT. Key finding: tokenization strategy and input phrasing have an outsized effect on numeric reasoning — more than model scale alone.

SLAM · 2026

SLAM in degenerate environments

Geometric SLAM degenerates in featureless or repetitive environments — long corridors, symmetric rooms — where point cloud registration loses constraint and drift accumulates unboundedly without loop closure. Built on SuperOdom, this project evaluates classical sensor fusion (LiDAR + IMU + wheel odometry) on real long-corridor robot trajectories to quantify how far multi-modal input alone can push drift bounds before a learned or topological component becomes necessary.

VLA · 2026

Cross-Embodiment Transfer for 5-DoF Manipulation

Most VLA and manipulation policies are trained and fine-tuned for 6-DoF arms. This project stress-tests cross-embodiment transfer by benchmarking those policies on a 5-joint xArm — a configuration they were never optimised for — across both simulation and real-world tasks. The goal is to characterise the performance gap and establish how much target-domain data is needed to close it via fine-tuning.

RL · 2026

Flappy Bird — Learning to Play with RL

A hands-on implementation of core RL algorithms — DQN, PPO, and policy gradient — applied to Flappy Bird. Built from scratch to understand how agents learn from sparse rewards, how hyperparameters affect convergence, and where each algorithm breaks down. A practical sandbox for getting intuition on RL before applying it to harder problems.
